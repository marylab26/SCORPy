# SCORPy: Single-Cell proteOmics Research Platform 

A platform for analyzing Cyclic Immunofluorescence (CycIF) or CO-Detection by indEXing  (CODEX) data with an intuitive web-based interface.

## Quick Start (Pre-built Binary)

1. Download the latest release from the [releases page](https://github.com/marylab26/SCORPy/releases)
2. Run the executable:
   - **Windows**: `SCORPy.exe`
   - **macOS/Linux**: `./SCORPy`
